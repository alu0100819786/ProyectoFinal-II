# ProyectoFinal-II


# Funcionamiento

  La aplicación trata de un Survival Shooter, donde tendremos que ir avanzando por un mapa infestado de enemigos acabando con ellos para conseguir la mayor puntuación posible.
  
  Moveremos a nuestro personaje con las teclas de control habituales WASD y rotaremos su cuerpo con QE.
  Para hacer uso de nuestra arma y poder acabar con nuestro enemigos deberemos pulsar la barra espaciadora y apagaremos y encenderemos nuestra linterna con la tecla L
  Además para hacer uso de la función de VR y poder realizar un control de la cámara deberemos pulsar Alt + Mover el ratón dentro de la pantalla de juego.
